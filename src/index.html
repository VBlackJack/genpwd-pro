GenPwd Pro

Version 3.0.5

Professional password manager with encrypted vault

Secure Vault

  • AES-256 Encryption with Argon2id for key derivation
  • Windows Hello: Biometric unlock (fingerprint, face)
  • Auto-lock: Automatic lock after inactivity
  • External Storage: OneDrive, Dropbox, custom locations

Advanced Generator

  • 3 modes: Syllables, Passphrase, Leet speak
  • Multi-language: French, English, Latin (2400+ words)
  • Entropy: Up to 140 bits of security
  • Presets: Savable and syncable configurations

Features

  • Import/Export: CSV compatible with other managers
  • Organization: Folders, tags, and advanced search
  • TOTP: 2FA code support (Google Authenticator)
  • Multi-platform: Web, Windows, macOS and Linux coming soon

Technology Stack

Electron ES6 Modules TweetNaCl Argon2id Web Crypto API

Keyboard Shortcuts

  • Alt + G Generate passwords
  • Alt + C Copy all passwords
  • Alt + R Run security tests
  • Alt + S Export results
  • Escape Close modals
